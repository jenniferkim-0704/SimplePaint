# (C# 코딩) 그림판 앱

## 개요
- C# 프로그래밍 학습
- 1줄 소개: 도형을 그릴 수 있는 그림판 앱
- 사용한 플랫폼: 
  -C#, .NET Windows Forms, Visual Studio, GitHub
- 사용한 컨트롤:
  - Label, Button, ComboBox, TrackBar, GroupBox, PictureBox
- 사용한 기술과 구현한 기능:
  - Visual Studio를 사용해 기본 UI배치 및 선택 기능 구현
