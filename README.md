# (C# 코딩) 그림판 앱

## 개요
- C# 프로그래밍 학습
- 1줄 소개: 직선, 사각형, 원을 그릴 수 있는 그림판 프로그래밍
- 사용한 플랫폼: 
  -C#, .NET Windows Forms, Visual Studio, GitHub
- 사용한 컨트롤:
  - Label, Button, ComboBox, TrackBar, GroupBox, PictureBox
- 사용한 기술과 구현한 기능:
  - Visual Studio를 사용해 기본 UI배치 및 선택 기능 구현
  - 마우스 드래그를 이용해 선택한 도형, 색, 선두께에 따라 직선, 사각형, 원 그리기 기능 구현

## 실행 화면 (과제2)
- 코드의 실행 스크린샷과 구현 내용 설명

![실행화면](img/screenshot-2.gif)

- 구현한 내용 (위 그림 참조)
  - 마우스 버튼을 누른 위치를 시작점으로 설정하고, 드래그한 위치를 기준으로 도형을 그리도록 구현
  - 변수 초기화 및 마우스 이벤트 처리: MouseDown, MouseMove, MouseUp 이벤트를 사용하여 도형 그리기 기능 구현
  - 드래그 중에는 현재 마우스 좌표를 계속 갱신하여 점선 미리보기 형태로 도형이 표시되도록 구현
  - 마우스 버튼을 떼면 끝점을 확정하고, Paint 이벤트를 사용하여 선택한 도형을 PictureBox에 최종적으로 그려 저장되도록 구현
  - 선택한 도형 종류(직선, 사각형, 원)에 따라 각각 다른 방식으로 도형을 그리도록 구현
  - 선택한 색상과 선 굵기가 실제 도형 그리기에 적용되도록 구현
